# Java Practice Problems

This directory contains practice exercises organized by difficulty level.

## How to Use

1. Read the problem description
2. Try to solve it on your own first
3. Run and test your code
4. Compare with the solution (if provided)
5. Understand different approaches

## Problem Categories

### Beginner Level
- Basic syntax and operations
- Simple calculations
- Input/Output operations
- Basic control flow

### Intermediate Level
- Loops and patterns
- Arrays and strings
- Methods and functions
- Object-oriented basics

### Advanced Level
- Complex algorithms
- Data structures
- File handling
- Exception handling

## Tips for Practice

1. **Start Simple**: Begin with easy problems
2. **Code Daily**: Practice consistently (30 mins daily)
3. **Debug**: Learn to read and fix errors
4. **Optimize**: After solving, think of better approaches
5. **Test**: Try different inputs to test your code
6. **Comment**: Write comments to explain your logic

## Running Java Programs

```bash
# Compile
javac ProgramName.java

# Run
java ProgramName
```
